ban：什么都不使用（水稻处于劣势，消退）
chem:继续使用化学品（不稳定，注意到在220周附近已经出现了一季的歉收）
green：改用生态手段（稳定）


产出与经济效益
effective开头的value和yield描述系统每季的产出。
考虑chem和green两种情况（由于这两种情况前两年的情况一致，所以产出数据也重合，可以截掉）

yield 的每行
1. chem模式下每季的水稻产出（取30%生物量）
2. green模式下每季的水稻产出（取30%生物量）
3. chem模式下每季的鸭产出（取100%生物量）
4. green模式下每季的鸭产出（取100%生物量）

value的两行描述chem和green模式下产出的价值（以水稻每千克0.5USD，鸭每千克5USD计）


稳定性
disaster开头的描述生态系统应对破坏恢复原状的能力。
采取green模式，考虑两种自然灾害，均发生在第145周：
1.flood 生产者的数目大幅减少
2.swarm insect的数目大幅增加
能够看出生态系统可以在几周内迅速条件恢复原状，故系统具有稳定性
（不做化学调节的是因为那个恢复更快，和我们的假设矛盾，不过我们可以这样说：因为化学农业需要化学药品的持续加入，一旦离开化学药品就会崩溃，所以其稳定性低于生物农业

系统对初值和参数的敏感程度
change开头的描述系统设置不同产生的影响。
仍采用green模式，取样80周。

init：研究系统对初值的敏感程度
第1-3行：系统所有生物量的初值分别取原值，原值的5倍，原值的1/5；
发现三者在经过一定时间后几乎重合，即模型对初值不敏感

parameters：研究系统对参数的敏感程度
第1-3行：水稻自然增长率分别取原值，原值的1.2倍，原值的0.8倍；
发现三者在经过一定时间后出现较大差异，即模型对参数敏感
